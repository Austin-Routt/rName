This is a random American name generator, which allows the user to choose the number of names he/she wants to generate, up to 100. What makes this particular random name generator special is that it relies on name frequency data obtained via the U.S. Census Bureau and Social Security Agency.

Unlike most random name generators, this data set allows one to take into account a weighted distribution of American first and last names when performing a random sampling by gender. Since the first name data contains the probabilities of names based on the occurrences of U.S. baby names from 1880 to 2013, and the surname data has frequencies based on the 2000 census, one need only reference the probabilities of these names when taking a sample. Thus, the random names you generate should be representative of an average American population, circa 2000; for additional information please click on the Paper tab on top of the main panel.

Instructions

To use the generator, simply click on the Name Generator tab, on top of this side panel, and specify the number of names that you wish to generate by using the slider. After you have made your choice, you should see all names displayed in the main panel under the Names tab.
